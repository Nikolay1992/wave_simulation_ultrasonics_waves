## 2. –§–∏–∑–∏–∫–æ-–º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –º–æ–¥–µ–ª—å

### 2.1. –û—Å–Ω–æ–≤–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ (SLS)

–î–ª—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å–º–µ—â–µ–Ω–∏—è $u_y(x,y,z,t)$ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –º–æ–¥–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –≤–æ–ª–Ω–æ–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ:

$\frac{\partial^2 u}{\partial t^2} = c^2(\mathbf{r}) \nabla^2 u + \eta(\mathbf{r}) \frac{\partial}{\partial t} \left( \nabla^2 u \right),
\quad \text{–≥–¥–µ} \quad
\eta(\mathbf{r}) = \frac{c^2(\mathbf{r})}{\omega_0 Q(\mathbf{r})}.$

- $\omega_0 = 2\pi f_0$ ‚Äî —É–≥–ª–æ–≤–∞—è —á–∞—Å—Ç–æ—Ç–∞ —Ü–µ–Ω—Ç—Ä–∞ –∏–º–ø—É–ª—å—Å–∞ ($f_0 = 1~\text{–ú–ì—Ü}$),
- $Q(\mathbf{r})$ ‚Äî –ª–æ–∫–∞–ª—å–Ω–∞—è –¥–æ–±—Ä–æ—Ç–Ω–æ—Å—Ç—å,
- $c(\mathbf{r})$ ‚Äî –ª–æ–∫–∞–ª—å–Ω–∞—è —Å–∫–æ—Ä–æ—Å—Ç—å –∑–≤—É–∫–∞.

> ‚úÖ –î–∞–Ω–Ω–∞—è —Ñ–æ—Ä–º–∞ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–∞ SLS –≤ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–∏–∏ **—á–∞—Å—Ç–æ—Ç–Ω–æ-–Ω–µ–∑–∞–≤–∏—Å–∏–º–æ–≥–æ $Q$** –≤ –æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏ $f_0$.

## 3. –ß–∏—Å–ª–µ–Ω–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è

### 3.1. –ú–µ—Ç–æ–¥ –∫–æ–Ω–µ—á–Ω—ã—Ö —Ä–∞–∑–Ω–æ—Å—Ç–µ–π (FDTD)

- –Ø–≤–Ω–∞—è —Å—Ö–µ–º–∞ **2-–≥–æ –ø–æ—Ä—è–¥–∫–∞** –ø–æ –≤—Ä–µ–º–µ–Ω–∏ –∏ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤—É.
- –ò—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–≤–∞ –ø–æ–ª—è:  
  - $u$ ‚Äî —Å–º–µ—â–µ–Ω–∏–µ,  
  - $v = \partial u / \partial t$ ‚Äî —Å–∫–æ—Ä–æ—Å—Ç—å.
- –õ–∞–ø–ª–∞—Å–∏–∞–Ω –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—ã–º–∏ —Ä–∞–∑–Ω–æ—Å—Ç—è–º–∏.
- –í—Ä–µ–º–µ–Ω–Ω–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è $\partial_t (\nabla^2 u)$ –∞–ø–ø—Ä–æ–∫—Å–∏–º–∏—Ä—É–µ—Ç—Å—è:  
  $\frac{\partial}{\partial t}(\nabla^2 u)^n \approx \frac{\nabla^2 u^n - \nabla^2 u^{n-1}}{\Delta t}.$

### 3.2. –£—Å–ª–æ–≤–∏—è —É—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç–∏

–í—Ä–µ–º–µ–Ω–Ω–æ–π —à–∞–≥ $\Delta t$ –≤—ã–±–∏—Ä–∞–µ—Ç—Å—è –∫–∞–∫ –º–∏–Ω–∏–º—É–º –∏–∑ –¥–≤—É—Ö —É—Å–ª–æ–≤–∏–π:

1. **–ö—É—Ä–∞–Ω—Ç–∞ (–¥–ª—è –≤–æ–ª–Ω–æ–≤–æ–≥–æ —á–ª–µ–Ω–∞)**:
   $\Delta t_{\text{CFL}} = \frac{0.75}{c_{\max} \sqrt{\frac{1}{\Delta x^2} + \frac{1}{\Delta y^2} + \frac{1}{\Delta z^2}}}.$

2. **–í—è–∑–∫–æ—Å—Ç–Ω–æ–µ (–¥–ª—è $\eta \partial_t \nabla^2 u$)**:
   $\Delta t_{\text{visc}} = 0.75 \cdot \frac{(\Delta x)^2}{2 \eta_{\max}}.$

‚Üí –ò—Ç–æ–≥–æ–≤—ã–π —à–∞–≥: $\Delta t = \min(\Delta t_{\text{CFL}},\ \Delta t_{\text{visc}})$.

## 4. –†–µ–∑—É–ª—å—Ç–∞—Ç—ã –∏ –ø–æ—Å—Ç–æ–±—Ä–∞–±–æ—Ç–∫–∞

### 4.1. –í—ã—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ

### 4.2. –û—Ü–µ–Ω–∫–∞ –¥–æ–±—Ä–æ—Ç–Ω–æ—Å—Ç–∏
